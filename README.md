# Earbud
Single Page Web App for browsing David Bowie's songs.

Data from [iTunes Search API](https://affiliate.itunes.apple.com/resources/documentation/itunes-store-web-service-search-api/)

Tech Stack:
* ASP.NET Core
* Angular 9
* Bootstrap 4
